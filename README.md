Mini Vector Search Engine

A simple AI-based search system that finds results based on meaning, not exact words.
Unlike traditional keyword search, this project uses vector embeddings to find
results.

---

 Key Idea

* Text → converted into vectors (numbers)
* Search → finds similar meaning, not same keywords

---

 Tech Used

* Python
* ChromaDB

---

How to Run

pip install chromadb
python main.py


---

Concepts Used

Vector embeddings
Semantic search
Cosine similarity (internally)
Vector databases
 Example

Input:
I like animals

Output:
I love dogs
Cats are cute animals
